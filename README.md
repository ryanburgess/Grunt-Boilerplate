Grunt Boilerplate
=============
A Grunt boilerplate to start your next project.
 
## Install
```js
npm install grunt-boilerplate -g
```

## CLI
Once you've installed grunt-boilerplate you can then run the command `grunt-boilerplate` to add Grunt to your project.

```js
grunt-boilerplate
```
